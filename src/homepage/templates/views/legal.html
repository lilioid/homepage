Summary

In short, neither this website nor any programs associated with it perform any sort of tracking. Metadata is also only stored for a very small amount of time and never sold nor automatically analyzed.

Scope of Application

This policy applies to pages that explicitly link to it and are hosted on one of the following domains or any of their subdomains:

  1. ftsell.de
  2. finn-thorben.me
Additionally it applies to emails sent to and received from any addresses ending with the following domains:
  1. @ftsell.de
  2. @finn-thorben.me
  3. @timonengelke.de
  4. @bit-bots.de
  5. @sheaksadi.me
  6. @bastianpermesang.com
  7. @powolygon.gay

Collected Information

Web Services

The webserver temporarily stores access and error logs which include details about a users request such as the client IP-Addresse, and which webpage was requested. The logs are purged regularly and never stored for longer than 48 hours.

The detailed nginx log format configuration can be seen below. Refer to the NGINX Documentation for details about it.

{"time": "$time_iso8601", "host": "$host", "status": "$status", "remote_addr": "$remote_addr", "request": "$request", "proxy_upstream": "$proxy_upstream_name", "request_length": "$request_length", "request_time": "$request_time" }

E-Mails

The mail server handling mails for the above mentioned domains which generates connection log entries when it receives E-Mails or sends E-Mails. The log contains information about the IP-Addresses and hostnames (determines via reverse-dns as well as SMTP-HELO) of connected clients as well as metadata about transferred mails. This metadata is limited to Recipient and Sender address of the mail, the content of the SMTP RCPT TO command and a unique ID per mail.

Purpose of Collection

The information collected is used to troubleshoot server errors and create IP or domain blocks when deemed necessary. The data is only stored on this server and will never be shared with any third party.
